# Propel PMO – AI Driven PMO Framework

This repository contains frameworks, templates, and dashboards for running an AI-enabled Program Management Office (PMO).

## Areas Covered

• AI PMO Operating Model  
• AI Governance Framework  
• Project Delivery Framework  
• Program Dashboards  
• Risk and Compliance Templates  
• AI Project Examples  

## AI PMO Model

The Propel PMO model integrates:

1. Strategy Layer
2. Governance Layer
3. Delivery Layer
4. Technology Layer
5. Operations Layer

## Example Projects

- AI Warehouse Optimization
- AI Chatbot QA Framework
- AI Program Dashboard

## Purpose

This repository demonstrates how modern PMOs can integrate AI capabilities, automation, and analytics to improve delivery performance.
